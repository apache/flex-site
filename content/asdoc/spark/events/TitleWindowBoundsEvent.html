Property Detail
afterBoundsproperty
public var afterBounds:Rectangle

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

The bounds of the object after the action. If this event is cancelable, afterBounds is the future bounds of the window. Otherwise, it is the current bounds.

The following list shows how this property is set for the different event types:

  • WINDOW_MOVE - The current bounds of the window.
  • WINDOW_MOVE_END - The final bounds of the window, before the drag.
  • WINDOW_MOVE_START - null.
  • WINDOW_MOVING - The future bounds of the window if the window moves to the current cursor position.

beforeBoundsproperty 
public var beforeBounds:Rectangle

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

The bounds of the object before the action. If this event is cancelable, beforeBounds is the current bounds of the window. Otherwise, it is the bounds before a change occurred.

The following list shows how this property is set for the different event types:

  • WINDOW_MOVE - The previous bounds of the window.
  • WINDOW_MOVE_END - The starting bounds of the window, before the drag.
  • WINDOW_MOVE_START - The starting bounds of the window.
  • WINDOW_MOVING - The current bounds of the window.

Constructor.

Parameters
type:String — The event type; indicates the action that caused the event.
 
bubbles:Boolean (default = false) — Specifies whether the event can bubble up the display list hierarchy.
 
cancelable:Boolean (default = false) — Specifies whether the behavior associated with the event can be prevented.
 
beforeBounds:Rectangle (default = null) — The bounds of the window before the action. If this event is cancelable, beforeBounds is the current bounds of the window. Otherwise, it is the bounds before a change occurred.
 
afterBounds:Rectangle (default = null) — The bounds of the window after the action. If this event is cancelable, afterBounds is the future bounds of the window. Otherwise, it is the current bounds.

The TitleWindowBoundsEvent.WINDOW_MOVE constant defines the value of the type property of the event object for a windowMove event.

The properties of the event object have the following values:

PropertyValue
bubblesfalse
cancelablefalse
currentTargetThe Object that defines the event listener that handles the event. For example, if you use myButton.addEventListener() to register an event listener, myButton is the value of the currentTarget.
beforeBoundsThe previous bounds of the object.
afterBoundsThe current bounds of the object.
targetThe Object that dispatched the event; it is not always the Object listening for the event. Use the currentTarget property to always access the Object listening for the event.
