Implementors MenuBarItem

Language Version : ActionScript 3.0
Product Version : Flex 3
Runtime Versions : Flash Player 9, AIR 1.1

The IMenuBarItemRenderer interface defines the interface that an item renderer for the top-level menu bar of a MenuBar control must implement. The item renderer defines the look of the individual buttons in the top-level menu bar. To implement this interface, you must define a setter and getter method that implements the menuBar, menuBarItemIndex, and menuBarItemState properties.
