MetadataToolsMethods
 
Packageflex.lang.reflect.utils
Classpublic class MetadataTools
InheritanceMetadataTools Object

The MetadataTools contains methods that are responsible for the parsing and interpretation of XML metadata. It contains methods that are responsible for determing whether provided metadata contains a certain tag as well as methods used to obtain information from the metadata based on provided keys.
