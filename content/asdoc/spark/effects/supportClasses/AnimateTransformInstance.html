The AnimateTransformInstance class implements the instance class for the AnimateTransform effect. Flex creates an instance of this class when it plays a AnimateTransform effect; you do not create one yourself.

Property Detail
autoCenterTransformproperty
public var autoCenterTransform:Boolean

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

If autoCenterTransform is true, the transform center is recalculated as the effect progresses, updating to any changes in the width and height of the object. If the property is false, the transformCenter property is used instead.

initializedproperty 
public var initialized:Boolean = false

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

If true, this single instance of the transform-related effects has already been initialized. This property is used by AnimateTransform to prevent duplicate initialization of the instance when there are multiple transform effects feeding into this single instance.

The default value is false.

transformCenterproperty 
public var transformCenter:Vector3D

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

The center around which the transformations in this effect occur. In particular, rotations rotate around this point, translations move this point, and scales scale centered around this point. This property will be ignored if autoCenterTransform is true. If autoCenterTransform is false and transformCenter is not supplied, then the center of the target object is used.

addMotionPath()method
public function addMotionPath(newMotionPath:MotionPath, newEffectStartTime:Number = 0):void

Language Version : ActionScript 3.0
Product Version : Flex 4
Runtime Versions : Flash Player 10, AIR 1.5

Adds a MotionPath object to the set of MotionPath objects in this instance, with the start time relative to the outermost parent effect. If there is already a MotionPath object for this effect instance that is acting on the same property as the new MotionPath object, then the keyframes of the new MotionPath are simply added to the existing MotionPath.

Parameters

newMotionPath:MotionPath — New MotionPath object.
 
newEffectStartTime:Number (default = 0) — Start time, in milliseconds, of the new MotionPath object.
