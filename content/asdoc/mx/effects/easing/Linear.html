Starting with Flex 4.0, Apache recommends that you use the spark.effects.easing.Linear class as an alternative to this class.

The Spark effects provided as of Flex 4 use classes which implement the IEaser interface instead of the easing functions in classes like Linear for the earlier Flex 3 effects. To achieve the same functionality of the old Linear class, create a new instance of spark.effects.easing.Linear or call spark.effects.easing.Linear.getInstance() to get a singleton instance.


The Linear class defines easing functions to implement linear motion with Flex effect classes. For more information, see http://www.robertpenner.com/profmx.
