Language Version : ActionScript 3.0
Runtime Versions : Flash Player 10, AIR 1.5

Defines constants for specifying subscript or superscript in the baselineShift property of the TextLayoutFormat class. You can specify baseline shift as an absolute pixel offset, a percentage of the current point size, or the constants SUPERSCRIPT or SUBSCRIPT. Positive values shift the line up for horizontal text (right for vertical) and negative values shift it down for horizontal (left for vertical).
