The TabbedViewNavigatorButtonBarHorizontalLayout class is a layout specifically designed for the TabbedViewNavigator. The layout lays out the children horizontally, left to right.

The layout sizes children to equal sizes and to fit the parent width.

All children are set to the height of the parent.
